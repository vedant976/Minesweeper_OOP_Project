# Minesweeper OOP Project

A classic Minesweeper game implemented using Object-Oriented Programming principles in Java.  
This project focuses on clean class design, modular structure, and core game logic.

Project Overview
- Implements the traditional Minesweeper game
- Uses object-oriented concepts such as classes, objects, encapsulation, and abstraction
- Designed to be easy to understand, maintain, and extend

Features
- Random placement of mines on the board
- Revealing cells with adjacent mine count
- Recursive reveal for empty cells
- Flag and unflag functionality
- Game over condition when a mine is revealed
- Win condition when all safe cells are revealed

Gameplay Description
- The board consists of hidden cells
- Some cells contain mines
- Revealing a cell shows the number of adjacent mines
- Players must avoid mines and reveal all safe cells to win

Installation
- Install Java Development Kit version 8 or higher
- Clone the repository using Git
- Open the project in any Java IDE or editor

How to Run
- Navigate to the src directory
- Compile all Java files
- Run the main class to start the game

Project Structure
- src folder contains all source code
- Main class acts as the entry point
- Board class manages the grid
- Cell class represents each cell
- Additional classes handle game logic
